# ESP32 E-Paper Badge — VBAT Sensing 审核清单

## 1. 电路拓扑 ✅

```
BAT+ ──── R12(100kΩ) ──┬── R13(100kΩ) ──┬── GND
                        │                  │
                        ├── C21(100nF) ────┘
                        │
                    GPIO34 (ADC1_CH6)
                    "VBAT_SENSE"
```

拓扑正确：分压器 + 低通滤波。

## 2. 分压比计算 ✅

- R12 = R13 = 100kΩ
- 分压比 = R13 / (R12 + R13) = 100k / 200k = 0.5
- BAT+ = 4.2V → VBAT_SENSE = 4.2 × 0.5 = 2.1V
- BAT+ = 3.0V → VBAT_SENSE = 3.0 × 0.5 = 1.5V
- ESP32 ADC1 量程 (atten=11dB): 0 ~ 3.6V
- **2.1V < 3.6V → 安全余量 71%** ✅

## 3. 功耗影响 ✅

- 静态电流 = VBAT / (R12 + R13) = 4.2V / 200kΩ = 21μA
- 24h 耗电 = 21μA × 24h = 504μAh
- 103040 软包电池容量 1200mAh → 日耗占比 = 504/1200 = 0.042%
- 年耗 = 184mAh → 续航影响约 15%（如果需要进一步压低 deep sleep 电流，后续可改为高阻值分压或用 MOSFET 控制分压上电）
- **对 1200mAh 软包电池有可见续航影响，后续量产建议评估高阻值或受控分压方案** ⚠️

## 4. GPIO 引脚选择 ✅

- GPIO34 = ADC1_CH6
- GPIO34 是 input-only 引脚 → 适合 ADC 读取 ✅
- GPIO34 属于 ADC1（不受 WiFi 影响，ADC2 在 WiFi 开启时不可用）✅
- GPIO34 原理图中已分配给 ESP32 pin 24 (IO34) ✅
- VBAT_SENSE label 同时出现在分压器中点和 ESP32 IO34 附近 ✅

## 5. 滤波电容 ✅

- C21 = 100nF 与 R12//R13 形成低通滤波
- 截止频率 f = 1/(2π×RC) = 1/(2π×50kΩ×100nF) ≈ 32Hz
- 足以滤除高频噪声，保留 DC 电池电压 ✅

## 6. 组件位置 ✅

- R12 at (230, 260) — 电源管理区域
- R13 at (230, 270) — 紧邻电池 BT1/J3
- C21 at (245, 270) — 旁路到 GND
- 所有位于 Section C (Power Management) 区域内

## 7. 组件命名 ✅

- R12, R13 延续已有 R 序列 (R1~R11)
- C21 延续已有 C 序列 (C1~C20)
- 无命名冲突
