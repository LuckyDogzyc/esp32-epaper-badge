# ESP32 墨水屏胸牌 PCB 设计外包需求

## 一、项目概述

设计一款基于 ESP32 的 2.13 英寸三色墨水屏低功耗胸牌/桌面摆件。

### 核心功能
- ESP32-WROOM-32D 主控（WiFi + BLE）
- 2.13 英寸三色墨水屏（红/黑/白，SPI 驱动）
- 软包聚合物锂电池供电（103040，1200mAh），PH2.0 插头连接
- USB-C 充电 + 串口烧录（共用一个 USB-C 口）
- 支持无电池插入 USB-C 直接运行（USB 直供电）
- 超低功耗设计（deep sleep 续航 >1年）
- 电池电量 ADC 检测

### 关键约束
- **极低功耗优先**：deep sleep 电流目标 <20μA
- **体积最小化**：板子尽量小，能装进胸牌外壳
- **BOM 成本敏感**：单台目标 ¥35 以内
- **双面板即可**

## 二、电源路径设计（重要）

### 2.1 电源路径管理（新增）

使用肖特基二极管实现 USB/电池自动切换，支持无电池运行：

```
USB-C VBUS(5V) ──→ D3(SS14) ──┐
                                ├──→ VCC_RAIL → HT7333 VIN → 3.3V → 系统
电池 VBAT(3.7~4.2V) ──→ D4(SS14) ──┘
```

- **USB 插入，有电池**：VBUS(5V) > VBAT，D3 导通 D4 截止 → USB 供电 + TP4056 给电池充电
- **USB 插入，无电池**：VBUS(5V) → D3 → HT7333 → 3.3V，系统正常运行
- **USB 拔掉，有电池**：D4 导通 → 电池供电
- **USB 拔掉，无电池**：系统断电（正常行为）

### 2.2 新增器件

| 位号 | 器件 | 封装 | 数量 | 说明 |
|------|------|------|------|------|
| D3 | SS14 肖特基二极管 | SMA | 1 | USB 供电路径 |
| D4 | SS14 肖特基二极管 | SMA | 1 | 电池供电路径 |

### 2.3 USB-C 统一功能

USB-C 口同时承担三个功能，**不需要额外充电口**：
1. **供电**：VBUS(5V) → D3 → HT7333 → 3.3V（无电池也能跑）
2. **充电**：VBUS(5V) → TP4056 → 给电池充电
3. **数据**：D+/D- → CH340N → ESP32 烧录/调试

### 3.1 原理图

- 连线完成后需通过 ERC（Electrical Rules Check）无错误
- 导出 PDF 原理图

### 3.2 PCB 布局
- 双面板
- 板子尺寸：建议 ≤70mm × 50mm（可协商）
- 接口位置要求：
  - USB-C 口放在板子一侧短边
  - 墨水屏 FPC 接口放在板子另一侧短边
  - 电池接口（PH2.0）放在板子长边侧面或背面
  - RST/BOOT 按键放在板子边缘可触及位置
- 注意事项：
  - ESP32 天线区域下方不可铺铜、不可走线
  - USB 差分线 D+/D- 等长、尽量短
  - SPI 信号线尽量短（ESP32→墨水屏）
  - 模拟部分（ADC 分压）远离数字高频信号
  - 充电电路（TP4056）走线足够宽
  - 肖特基二极管 D3/D4 靠近电源入口放置

## 四、原理图连线需求（Pin-to-Pin）

### 4.1 电源网络

| 网络 | 连接 |
|------|------|
| **VBUS** | USB-C VBUS(J1) → FB1 → TP4056 VCC(4) → C6 正极 → D3 阳极 |
| **VBAT** | 电池正极 → J3 pin1 → TP4056 BAT(5) → D4 阳极 → R12 上端 → C1/C9/C11 正极 |
| **VCC_RAIL** | D3 阴极 + D4 阴极 汇聚 → HT7333 VIN(3) |
| **3V3** | HT7333 VOUT(2) → C8/C12 正极 → ESP32 VDD → 墨水屏 VCC → CH340 VCC(16) → 去耦电容正极 |
| **GND** | 所有组件 GND 汇聚 → J3 pin2 → USB-C GND → HT7333 GND(1) |

### 4.2 充电电路

| 网络 | 连接 |
|------|------|
| **VBUS→TP4056** | USB-C VBUS → FB1 → TP4056 VCC(4) → C6 正极 → CH340 VCC(16) |
| **CHRG** | TP4056 CHRG(1) → R4 → D1(LED红) 阳极 → D1阴极→GND |
| **STDBY** | TP4056 STDBY(3) → R5 → D2(LED绿) 阳极 → D2阴极→GND |
| **PROG** | TP4056 PROG(2) → R3 → GND (R3=1.2kΩ → 充电电流≈1A) |

### 4.3 电池保护

| 网络 | 连接 |
|------|------|
| **CS** | 电池正极 → R6(10k) → DW01A CS(3) → R7(100Ω) → GND → C10(100nF) |
| **OD** | DW01A OD(1) → 8205A G1(2) |
| **OC** | DW01A OC(3) → 8205A G2(4) |
| **BAT-** | 电池负极 → 8205A 源极/漏极 → TP4056 GND(3) |

### 4.4 ESP32 主控

| 网络 | 连接 |
|------|------|
| **EN** | ESP32 EN → R1(10k) → 3V3 → C10(100nF) → GND → SW1 → GND |
| **IO0** | ESP32 IO0 → R2(10k) → 3V3 → SW2 → GND |
| **TX** | ESP32 TX(GPIO1) → R8(1k) → CH340 RX(2) |
| **RX** | ESP32 RX(GPIO3) → R9(1k) → CH340 TX(3) |

### 4.5 墨水屏 SPI

| 网络 | 连接 |
|------|------|
| **EPD_MOSI** | ESP32 GPIO13 → J2 MOSI |
| **EPD_CLK** | ESP32 GPIO14 → J2 SCK |
| **EPD_CS** | ESP32 GPIO15 → J2 CS |
| **EPD_DC** | ESP32 GPIO27 → J2 DC |
| **EPD_RST** | ESP32 GPIO26 → J2 RST |
| **EPD_BSY** | ESP32 GPIO25 → J2 BUSY |

### 4.6 USB 转串口

| 网络 | 连接 |
|------|------|
| **D+** | USB-C D+(J1) → CH340 D+(5) |
| **D-** | USB-C D-(J1) → CH340 D-(6) |

### 4.7 电池电量检测

| 网络 | 连接 |
|------|------|
| **VBAT_SENSE** | R12(100k, VBAT端) → R13(100k, GND端) 中点 → C21(100nF)//R13 → ESP32 GPIO34(ADC1_CH6) |
